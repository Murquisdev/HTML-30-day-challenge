Crea un slider que permita seleccionar un número entre 1-50 y lo muestre en vivo al cambiar. Requiere algo de Javascript.

HTML tiene una serie de etiquetas que permiten introducir datos al usuario y que se suelen utilizar en formularios. Estas son las etiquetas input

Las etiquetas input tienen el atributo type con el que podemos restringir al usuario que datos pueden introducir.

En nuestro caso necesitamos que introduzca solo números, con lo que se pueden usar dos etiquetas type:

  • number: Permite introducir un número cualquiera
  • range: Permite introducir un número dentro de un rango (por defecto no muestra el valor elegido y el rango es entre 1 y 100)
    • output: Si se quiere mostrar el número se añade la etiqueta output justo después del input y se puede actualizar mediante el envento de JS onInput

También se puede limitar los valores a introducir con los atributos min y max

        
            <form method="post" action="/send/">
                Un valor numérico:
                <input type="number" value="1" min="1" max="50">
              
                Un valor numérico en rango:
                <input type="range" value="1" min="1" max="50"
                       onInput="this.nextElementSibling.value = this.value">
                <output>25</output>
            </form>
