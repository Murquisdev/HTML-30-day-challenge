Formas de añadir estilos CSS a una página web

Hay 3 formas de añadir código css a una página HTML

  • Estilos en línea
  • Directamente en la cabecera del documento HTML (bloque de estilos)
  • Desde un archivo externo

Estilos en línea

Se añaden directamente dentro de las etiquetas HTML utilizando el atributo style

        
            <p style="color:navy;">Este texto saldrá con color "navy""</p>

Este texto saldrá con color "navy""

Bloque de estilos

Se añaden las etiquetas style

No es obligatorio, pero se suelen poner en la cabecera del documento HTML (head), ya que era la forma antigua de hacerlo

Archivo externo

Se añade el enlace en la cabecera del archivo HTML aputando al archivo css

        <link rel="stylesheet" href="styles.css">

Se aconseja escribir esta línea lo antes posible (sobre todo, antes de los scripts), obligando así al navegador a aplicar los estilos cuanto antes y eliminar la falsa percepción visual de que la página está en blanco y no ha sido cargada por completo.
