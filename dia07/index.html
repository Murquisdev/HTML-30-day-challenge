Día 7

Coloca una imagen en formato JPEG-XL. Si el navegador no la soporta, que use AVIF. Sino, que use JPG.

Hay dos etiquetas HTML que nos permiten elegir que tipo de imagen se cargará en el navegador según los criterios que le indiquemos

Estas etiquetas son PICTURE y SOURCE

  • picture: Funciona a modo de contenedor, es donde indicaremos las diferentes imágenes a cargar
  • source: Funciona a modo de condición, el navegador irá comprobando por orden si puede cargar las imágenes.
        
            <picture>
                <source srcset="image.avif">
                <source srcset="image.webp">
                <img src="image.jpg" alt="Descripción breve de la imagen">
            </picture>

El navegador intentará cargar la imagen "image.avif", si no puede, pasa a "image.webp y por último a image.jpg

Lo que está haciendo el navegador es poner el atributo srcset en el atributo src de la etiqueta img

Source

La etiqueta source tiene diferentes atributos posibles

  • srcset: Serie de imágenes (separadas por coma) que se utilizarán. Atributo obligatorio.
  • src: Se utiliza cuando se indica audio o video. En imágenes es ignorado.
  • sizes: Consulta de medios (media queries) para determinar si la imagen o imágenes se mostrarán.
  • media: Condición que se debe cumplir para que muestre la imagen. Ver media queries.
  • type: Sugerencia para indicar al navegador el formato que se utiliza. Opcional.

En el siguiente ejemplo se mostrará una imagen según el ancho de la pantalla

        
            <picture>
                <source media="(width >= 900px)"
                        srcset="logo-large.png">
                <source media="(width >= 600px) and (width < 900px)"
                        srcset="logo-medium.png">
                <source media="(width < 600px)"
                        srcset="logo-small.png">
                <img src="logo-medium.png" alt="HTML5 logo">
            </picture>

En este ejemplo, además de comprobar el ancho de pantalla se comprueba la densidad de píxeles del dispositivo.

        
            <picture>
                <source media="(width >= 900px)"
                        srcset="logo-large.png,
                                logo-large-hd.png 2x,
                                logo-large-fhd.png 3x">
                <source media="(width >= 600px) and (width < 900px)"
                        srcset="logo-medium.png,
                                logo-medium-hd.png 2x,
                                logo-medium-fhd.png 3x">
                <source media="(width < 600px)"
                        srcset="logo-small.png,
                                logo-small-hd.png 2x,
                                logo-small-fhd.png 3x">
                <img src="logo-medium.png" alt="HTML5 logo">
            </picture>

Si no se pone nada, por defeco es 1x, pero si indicamos 2x, 3x... comprobará la densidad de píxeles del dispositivo.
