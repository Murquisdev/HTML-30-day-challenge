Para añadir contenido externo a una página web se usa la etiqueta IFRAME

        
            <iframe src="link del contenido" width="560" height="315"></iframe>

Tiene diferentes atributos:

  • name: Sirve para crear una referencia al marco y así poder cargar diferente contenido externo en él.
  • srcdoc: Permite añadir directamente código fuente (html) al iframe
  • allowfullscreen: Sirve para indicar si se permite poner el iframe en pantalla completa
  • loading: Indica la forma de cargar el iframe
    • eager: Por defecto, el contenido se dercaga inmediatamente
    • lazy: Se pospone la descarga

En el siguiente ejemplo, al hacer click en el enlace, se abrirá el buscador Bing en el iframe

        
            <a href="https://www.bing.com/" target="marco">Abrir buscador Bing</a>

            <iframe name="marco" width="100%" height="450"
                src="https://www.youtube.com/embed/Imeq3GeRttw">
            </iframe>

En el siguiente ejemplo, se mostrará en el contenido de srcdoc si puede cargarlo, sino cargará src

        
            <iframe src="http://pagina.com/apartado/"
                srcdoc="<p>¡Hola! ¡Soy el <strong>texto del iframe</strong>!</p>">
            </iframe>
