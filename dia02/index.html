¿Qué son los titulares o headers en HTML?

HTML tiene hasta 6 niveles de titulares

        
            <h1>Titular 1</h1>
            <h2>Titular 2</h2>
            <h3>Titular 3</h3>
            <h4>Titular 4</h4>
            <h5>Titular 5</h5>
            <h6>Titular 6</h6>

En esta página, Día 2 sería el título (h1), Página con titular, varios... sería el subtítulo (h2) y ¿Qué son los... sería la sección (h3) por ejemplo.

Insertar imágenes en HTML

Para insertar imágenes en HTML se emplea la etiqueta <img src="" alt="">

No se necesita la etiqueta de cierre e indicamos la dirección de la imagen en el atributo src="" y en alt="" se indicará un texto alternativo por si la imagen no carga o para los lectores de adaptabilidad.

Psyduck, el mejor Pokemon de todos 😁

También se le pueden añadir los atributos width y height para indicar el ancho y el alto en píxeles respectivamente

Se considera buena práctica indicar un width y height aunque luego se cambien los valore mediante css, porque reserva el espacio en la carga de la página y evita saltos molestos durante esta.

Más atributos de la etiqueta <img>

Loading:Indica si el navegador descarga inmediatamente la imagen o espera

  • loading="eager": Por defecto, descarga la imagen.
  • loading="lazy": El navegador espera para descargarla.

fetchpriority: Establece la prioridad de descarga de las imágenes

  • fetchpriority="auto": Por defecto, el navegador establece la prioridad
  • fetchpriority="high": Prioridad alta
  • fetchpriority="low": Prioridad baja

decoding: Indica al navegador como tiene que realizar el proceso de decodificación

  • decoding="auto": Por defecto, el navegador decide como hacer el proceso
  • decoding="sync": El navegador decodifica primero esta imagen antes que el resto
  • decoding="auto": El navegador decodifica primero otras imagenes ante que esta

srcset: Se utiliza para indicar diferentes imágenes según la resolución de pantalla y así evitar el consumo de recursos

  • Ancho: Para indicar el ancho de la pantalla en píxeles se utiliza la letra w (400w,800w,...)
  • Densidad: Para indicar la densidad de la pantalla en píxeles se utiliza la letra x (3x, 2.5x,...)
  • decoding="sync": El navegador decodifica primero esta imagen antes que el resto
  • decoding="auto": El navegador decodifica primero otras imagenes ante que esta
        
            <img src="imagen_400w.jpg" alt="Logotipo HTML5"
            srcset="imagen_400w.jpg 400w, imagen_900w.jpg 900w, imagen_1200w.jpg 1200w">

sizes: Se establecen unas condiciones según las media queries

        
            <img src="imagen_1200w.jpg" alt="Logotipo HTML5"
             srcset="imagen_1200w.jpg 1200w,
                     imagen_900w.jpg 900w,
                     imagen_400w.jpg 400w"
            sizes="(width > 900px) 900px,
                   ((width > 480px) and (width <= 800px)) 480px,
                   ((width <= 480px)) 320px">
