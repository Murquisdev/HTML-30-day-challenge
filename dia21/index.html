Crea una lista desplegable donde se pueda seleccionar entre 3 grupos de productos ficticios.

Seleciona los ingredientes:

Cuando tenemos demasiadas opciones entre las que escoger, se suele utilizar una lista desplegable, para ello utlizamos la etiqueta select. 

            
                <form method="post" action="/send/">
                    Selecciona la opción deseada:
                    <select>
                        <option value="1">Opción 1</option>
                        <option value="2" selected>Opción 2</option>
                        <option value="3">Opción 3</option>
                    </select>
                  </form>
Selecciona la opción deseada:

Cada opción se pone con la etiqueta option

Si queremos que una opción salga seleccionada por defecto, se utliza el atribuo selected

            
                <form method="post" action="/send/">
                    Selecciona la opción deseada:
                    <select>
                        <option value="1" disabled selected>-Elige una opción-</option>
                        <option value="1">Opción 1</option>
                        <option value="2" >Opción 2</option>
                        <option value="3">Opción 3</option>
                        <option value="4">Opción 4</option>
                        <option value="5">Opción 5</option>
                    </select>
                  </form>
Selecciona la opción deseada:

Puede ser interesante usar el atributo disable para usar una opción como placeholder.

Pero hay que tener cuidado, puesto al estar inicialmente seleccionada para mostrarla, no cumplirá con la restricción del formulario, porque hay algo seleccionado.

Grupos de opciones

Se pueden crear grupos de opciones y separarlos con las siguientes etiquetas:

  • hr: Muestra una línea dentro de la selección.
  • optgroup: Agrupa las opciones por grupos
            
                <form method="post" action="/send/">
                    Selecciona la opción deseada:
                    <select>
                      <optgroup label="Categoría 1">
                        <option value="1">Opción 1</option>
                        <option value="2">Opción 2</option>
                      </optgroup>
                      <optgroup label="Categoría 2">
                        <option value="3">Opción 3</option>
                        <option value="4">Opción 4</option>
                        <option value="5">Opción 5</option>
                      </optgroup>
                    </select>
                </form>

Las etiquetas optgroup no son seleccionables, pero si se les puede añadir el atributo disable para no permitir la selección

Selecciona la opción deseada:

También se puede permitir una selección múltiple añadiendo el atributo multiple a la etiqueta select

Hay que recordar que para realizar multiples selcciones hay que matener presionado la tecla CTRL o SHIFT
