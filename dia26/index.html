Día 26 - Murquisdev

Crea un mensaje emergente al pulsar un botón, que desaparezca al pulsar fuera del mensaje.

HTML5 tiene el atributo popover con el que se pueden crear elementos emergentes sin falta de JS:

  • popover: Convierte el elemento HTML en un potencial contenido emergente (auto por defecto)
  • popovertarget: Indica el id del elemento a manipular
  • popovertargetaction: Indica la acción a realizar
    • show: Lo muestra
    • hide: Lo oculta
    • toggle: Va cambiando

Por defecto el elemento con el atributo popover desparece. Esto es porque los navegadores tienen un estilo CSS que los oculta por defecto.

            
                <button popovertarget="message" popovertargetaction="toggle">Mostrar mensaje</button>
                <div id="message" popover>
                    <p><strong>¡Psyduck!</strong> es el mejor <em>Pokémon</em> de todos.</p>
                </div>

¡Psyduck! es el mejor Pokémon de todos.

Como siempre, con CSS se puede modificar el elemento emergente e incluso como se ve el fondo:

  • [popover]: Aplica los estilos a cualquier elemento con el atributo popover
  • ::backdrop: Aplica estilos al fondo de la página al mostrar el popover
  • :popover-open: Aplica estilos al popover abierto
