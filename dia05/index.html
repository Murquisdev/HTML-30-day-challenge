Día 5

Añadir un título y una descripción al documento, ideal para SEO

Un documento HTML se compone de dos partes bien diferenciadas, Head y Body

Etiquetas de la cabecera del documento (HEAD)

En la etiqueta HEAD se indican los metadatos, información sobre el documento (título, descripción...)y las relaciones con otros documentos (CSS, JS...)

  • <title>: Indica el título de la página en la pestaña del navegador o en los motores de búsqueda (Google por ejemplo)
  • <base>: URL base de los enlaces (usado para gestionar rutas relativas).
  • <link>: Establece una relación del documento actual con otro externo (CSS, JS...).
  • <style>: Crea estilos CSS que afectarán únicamente al documento actual. También se puede usar en el body
  • <script>: Indica un script a cargar o ejecutar en la página actual. También se puede usar en el body
  • <meta>: Establece un metadato específico en el documento actual.
Se recomienda que al menos siempre existan estas dos etiquetas: 
        
            <title>Título del documento</title>
            <meta charset="utf-8">
Además añadir una descripción ayuda al SEO y es la información que aparecerá en los buscadores cuando alguien encuentre tu página

Etiqueta META

Como norma general, la etiqueta META tendrá los atributos name y content.

Dependiendo del valor de name se indicará el valor de content

        
            <meta name="..." content="...">

La codificación de texto es muy importante para no encontrar caracteres raros o que el navegador no sabe interpretar

        
            <meta charset="utf-8">

El tipo de codificación más extendida y recomendada es utf-8

Etiquetas de metadatos y SEO

  • description: Indica la descripción de la página que aparece en buscadores (texto)
  • keywords: Lista de palabras clave separadas por comas. Google no la tiene en cuenta.
  • author: Indica el nombre del autor de la página.
  • application-name: Indica el nombre de la aplicación web. Debería usarse sólo si es una webapp.
  • generator: Indica el software utilizado para crear la página web.
  • theme-color: Indica el color (en HEXADECIMAL) con el que mostrar la barra del navegador

En la etiqueta meta description se añadirán unas frases a modo de resumen de la página, suele ser un texto entre 50 y 160 carácteres 

        
            <meta name="description" content="En esta página encontrarás documentación sobre HTML.">
            <meta name="keywords" content="html, lenguaje, lenguaje de marcado, código html, etiqueta">
            <meta name="author" content="Manz">
            <meta name="application-name" content="ManzDev Skynet">
            <meta name="generator" content="VSCode">
            <meta name="theme-color" content="#1e7bbf">
