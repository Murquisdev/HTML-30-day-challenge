Día 9

Crea un pergamino con una lista de objetos, donde se numere con números romanos (mayúsculas).

Para crear listas en HTML se utilizan las etiquetas UL (unorder list) para listas desordenadas y OL (order list) para listas ordenadas

Luego cada elemento de la lista tiene que estar entre las etiquetas li (list item)

A estos items se les puede modificar el símbolo que muestran, el número, añadir metadatos,... añadiendo atributos

  • value: Sirve para añdir un valor número como metadato, que se podrá utilizar en CSS o JS. Se mostrará ese dato si la lista es OL
  • start: Indica el número del primer ítem de la lista y del que empezará a contar.
  • type: Lista con números enteros, letras o números romanos (minúsculas o mayúsculas). (1 | a | A | i | I )
