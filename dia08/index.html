Día 8

Crear un párrafo de texto con palabras destacadas en diferentes colores.

Para resaltar texto se utiliza la etiqueta MARK

        
            <p>
                Este texto no está resaltado. <mark>Este sí está resaltado</mark>
            </p>

Es una etiqueta semántica que se utiliza para llamar la atención del lector y crea un efecto de subrayado.

El efecto de subrayado se puede modificar con CSS. Por ejemplo añadiendo una clase

        
            <p>
                <mark class="marcadoRojo">Texto subrayado de color rojo</mark>
            </p>
  • Subrayado rojo
  • Subrayado naranja
  • Subrayado verde

Así podemos crear un texto con diferentes subrayados.
