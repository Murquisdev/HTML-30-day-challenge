Crear un grupo de secciones (acordeón) donde se depliegue sólo uno a la vez.

Para poder crear elementos desplegables (o acordeón HTML) necesitamos la etiqueta DETAILS

Esta etiqueta actua como un DIV, salvo que su contenido se muestra inicialmente oculto.

            
                <details>
                <p>Esto es una información que en principio aparecerá oculta.</p>
                </details>

Se le puede añadir el atributo open para que se muestre el contenido sin falta de hacer click

Por defecto, la etiqueta DETAILS añade el título "detalles", aunque depende del idioma del navegador

Este título se puede cambiar añadiendo la etiqueta SUMMARY dentro de la etiqueta DETAILS

Cambiar el título del desplegable 
            
                <details>
                    <summary>Cambiar el título del desplegable</summary>
                    <p>información a mostrar</p>
                </details>

Si tenemos varios acordeones y le asignamos el mismo atributo name, conseguimos que solo se muestre un acordeón abierto y el resto se cierren, sin necesidad de JS.

        
            <details name="acordeon">
                <summary>Cambiar el título del desplegable</summary>
                <p>información a mostrar</p>
            </details>
