Día 27 - Murquisdev

Crea una serie de etiquetas que permitan mostrar como miniatura una imagen en redes sociales.

En el head del documento HTML se establecen una serie de etiquetas meta.

Además de las etiquetas vistas (title, description), se pueden añadir etiquetas con información para redes sociales.

Facebook (Meta)

Meta utilza el protocolo Open Graph:

            
                <head>
                    <meta property="og:type" content="article">
                    <meta property="og:title" content="Título del post">
                    <meta property="og:url" content="https://pagina.com/posts/32/titulo-del-post">
                    <meta property="og:image" content="https://pagina.com/assets/post-32.jpg">
                    <meta property="og:description" content="Descripción del post de la página">
                </head>

Se puede utilizar el Open Graph Debbuger para comprobar errores.

Twitter (X)

Por su lado Twitter utiliza otro sistema llamado Twitter Cards

            
                <head>
                    <meta property="twitter:card" content="summary_large_image" />
                    <meta property="twitter:creator" content="@Manz" />
                    <meta property="twitter:title" content="Título" />
                    <meta property="twitter:url" content="https://pagina.com/posts/32/titulo-del-post">
                    <meta property="twitter:image" content="https://pagina.com/assets/post-32.jpg" />
                    <meta property="twitter:description" content="Descripción" />
                </head>

Tiene el Twitter Cards Validator, para comprobar errores.

Se recomienda un formato de imagen 2:1, con un tamaño mínimo de 300x157 y máximo de 4096x4096 (5MB) (JPG, PNG, WEBP o GIF)

Quedando la cabecera del documento como sigue para que las redes sociales de Meta y Twitter puedan crear una previsualización:

            
                <!-- Etiquetas de Facebook (Meta) -->
                <meta property="og:type" content="article">
                <meta property="og:title" content="Solución del reto del Día 27 del HTML-30-day-challenge de ManzDev por Murquisdev">
                <meta property="og:url" content="https://murquisdev.github.io/HTML-30-day-challenge/dia27/index.html">
                <meta property="og:image" content="https://murquisdev.github.io/HTML-30-day-challenge/dia27/img/psyduck3.jpg">
                <meta property="og:description" content="Solución del reto del Día 27 del HTML-30-day-challenge de ManzDev por Murquisdev. Crea una serie de etiquetas que permitan mostrar como miniatura una imagen en redes sociales.">
                <!-- FIN Etiquetas de Facebook (Meta) -->
                <!-- Etiquetas de Twitter (X) -->
                <meta property="twitter:card" content="summary_large_image">
                <meta property="twitter:creator" content="@murquisdev">
                <meta property="twitter:title" content="Solución del reto del Día 27 del HTML-30-day-challenge de ManzDev por Murquisdev">
                <meta property="twitter:url" content="https://murquisdev.github.io/HTML-30-day-challenge/dia27/index.html">
                <meta property="twitter:image" content="https://murquisdev.github.io/HTML-30-day-challenge/dia27/img/psyduck3.jpg">
                <meta property="twitter:description" content="Solución del reto del Día 27 del HTML-30-day-challenge de ManzDev por Murquisdev. Crea una serie de etiquetas que permitan mostrar como miniatura una imagen en redes sociales.">
                <!-- FIN Etiquetas de Twitter (X) -->
