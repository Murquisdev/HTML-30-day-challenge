Día 3

Construye una página con un párrafo que tenga enlances internos y externos

Enlaces

  • Atributo href
  • Atributo hreflang
  • Atributo download
  • Atributo target
  • Atributo rel
  • Atributo rel
  • Atributo Enlaces internos

Los enlances son la base de internet, nos permiten navegar, es decir, movernos entre páginas, pero también sirven para moverse en la misma página.

        
            <a href="https://direccionweb.com">Web</a>

En el atributo href se indica la dirección web de destino.

Por ejemplo, voy a enlazar la página web de Manz en el siguiente texto: Página web de Manz.

El atributo hreflang se indica el idioma mediante un código ISO 639-1(Los países en siglas de dos caracteres).

El atributo download se indica que es un archivo de descarga.

        
            <a href="/productos.pdf" download>Descargar productos</a>

El atributo target se indica donde se va abrir el enlace, por defecto en la misma ventana, pero añadiendo _blank, se abrirá en otra pestaña

        
            <a href="https://manz.dev" target="_blank">Página web de Manz</a>

Otros valores del atributo target son:

  • _self: Abre en la pestaña o iframe actual
  • _parent: Abre en el documento padre
  • _top: Abre en el documento raíz (padre global)
  • nombre: Abre en el iframe con ese nombre

El atributo rel: Indicamos la relación con el contenido del enlace.

  • alternate: Indica que el enlace es una versión alternativa del documento actual.
  • author: Indica que es el enlace de la web del autor del documento actual.
  • bookmark: El enlace es un permalink de la sección padre más cercana.
  • help: Indica que el enlace ofrece ayuda relacionada con la página actual.
  • license: Indica que la página actual está cubierta por la licencia referenciada.
  • nofollow: Indica que el enlace no está supervisado por el autor del sitio web.
  • next: Indica que el enlace es la parte siguiente del documento actual.
  • noreferrer: El navegador no envía la página de procedencia al visitar el enlace.
  • search: Indica que el enlace es una página para realizar búsquedas.
  • tag: Asocia una etiqueta al enlace especificado, en relación al documento actual.
  • external: Indica que el enlace no forma parte del sitio actual (recurso externo).

El atributo type: Indicamos el tipo de recurso que está enlazando

        
            <a type="application/pdf" href="/prices.pdf">Examinar fichero PDF con lista de precios</a>

Los tipos se indican siguiendo la nomenclatura tipo MIME (tipo de medio), que se compone de dos partes separadas por /

Para poner enlaces dentro de la misma pagina necesitamos identificar de alguna forma donde queremos ir.

Para ello nos ayudamos del atributo id, con lo que podemos indicar al atributo href la dirección interna añadiendo el caracter #

De esta forma es posible navegar dentro de la misma página, por ejemplo como el índice de esta página

        
            <a href="#indice">índice</a>
