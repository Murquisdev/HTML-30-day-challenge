En la lista anterior, permite al usuario introducir opciones personalizadas (no sólo las de la lista) y filtrar las opciones existentes.

Seleciona el ingrediente:

La etiqueta datalist se utiliza cuando queremos dar unas opciones a elegir, además de que el usuario quiera añadir las suyas.

            
                <datalist id="items">
                    <option value="1">Opción 1</option>
                    <option value="2">Opción 2</option>
                </datalist>

Esto no mostrará nada visualmente, porque solo crea la lista de datos, ahora hay que conectarla a un campo de entrada.

Por ejemplo: Para conectar el datalist con el input, utilizamos el atributo list y le agregamos el valor del #id del datalist

            
                <form method="post" action="/send/">
                    <input type="text" list="items">
                    <datalist id="items">
                      <option value="manzdev">ManzDev</option>
                      <option value="afor_digital">Afor_digital</option>
                      <option value="padawanstrainer">PadawansTrainer</option>
                      <option value="chrisvdev">ChrisvDev</option>
                    </datalist>
                </form>

En este ejemplo, vemos que el id del datalist es items y al atributo list del input le hemos agregado el valor items. Asociando así la lista a ese input

Si el usuario escribe, se filtraran los resultados a mostrar. En Firefox es necesario hacer doble click para mostrar los elementos.

Con lo que el usario podrá introducir su propio valor o alguno ya existente.
