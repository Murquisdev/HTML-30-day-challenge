Carga una librería Javascript de confetti y lánzalo cuando pulses en un botón. Requiere algo de Javascript.

En el reto del día 14 ya se vio como cargar un script

La librería a utlizar será: js-confetti

Como puse en funcionamiento la librería:

  1. Copié el código JS de la dirección https://cdn.jsdelivr.net/npm/js-confetti@latest/dist/js-confetti.browser.js.
  2. Cree el fichero js-confetti.js y pegué el código.
  3. Hice que la función que hay en js-confetti.js fuera exportable
  4. La importé en index.js
  5. Hice el proceso de recuperar el botón, añadir el "escuchador" y crear la función que realiza la acción.
            
                import {JSConfetti} from "./js-confetti.js" // importo la librería

                // Selecciono el elemento a interaccionar
                const confettiButton = document.querySelector("button")

                //Añado los eventos
                confettiButton.addEventListener("click", () => launchConfetti())

                // Funcion
                function launchConfetti (){
                const jsConfetti = new JSConfetti() // Creamos el confetti
                jsConfetti.addConfetti() //  Lanzamos el confetti
                }
